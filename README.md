# viz3d

Простая Python-библиотека для интерактивной 3D-визуализации математических функций. Запускает локальный сервер и открывает график в браузере (на базе Plotly).

## Установка

```bash
cd viz3d
pip install -e .
```

Зависимости: `numpy`, `flask`. Plotly подгружается в браузере с CDN.

## Использование

### Вариант 1 — Поверхность z = f(x, y) (2 параметра на вход, 1 на выход)

```python
from viz3d import plot
import numpy as np

# Параболоид
plot(lambda x, y: x**2 + y**2)

# Седло
plot(lambda x, y: x**2 - y**2)

# Волны
plot(lambda x, y: np.sin(np.sqrt(x**2 + y**2)))
```

### Вариант 2 — Параметрическая поверхность (x, y, z) = f(u, v)

Функция принимает 2 параметра и возвращает кортеж из 3 значений — точку в пространстве.

```python
from viz3d import plot
import numpy as np

# Сфера
plot(
    lambda u, v: (np.cos(u) * np.sin(v), np.sin(u) * np.sin(v), np.cos(v)),
    x_range=(0, 2 * np.pi),
    y_range=(0, np.pi),
)

# Тор
def torus(u, v, R=2, r=1):
    return (
        (R + r * np.cos(v)) * np.cos(u),
        (R + r * np.cos(v)) * np.sin(u),
        r * np.sin(v),
    )

plot(torus, x_range=(0, 2 * np.pi), y_range=(0, 2 * np.pi))
```

## Параметры функции `plot`

| Параметр       | По умолчанию | Описание                                  |
|----------------|--------------|-------------------------------------------|
| `func`         | —            | Функция от 2 параметров                   |
| `x_range`      | `(-5, 5)`    | Диапазон первого параметра                |
| `y_range`      | `(-5, 5)`    | Диапазон второго параметра                |
| `resolution`   | `80`         | Число точек по каждой оси                 |
| `port`         | авто         | Порт сервера                              |
| `open_browser` | `True`       | Автоматически открыть браузер             |
| `title`        | авто         | Заголовок графика                         |

Тип графика (поверхность или параметрическая) определяется автоматически по тому, что возвращает функция.
